## About

ImagePro is a free photo editing application designed for users of any skill level. It combines an intuitive interface with professional tools such as adjusting brightness, contrast, applying filters, and editing with a brush. The application supports the main image formats (JPG, PNG, BMP), provides user registration and authorization, and is also convenient for use by both amateurs and professionals.

Technology stack: C#, Windows Forms, SQL Server, Visual Studio.
